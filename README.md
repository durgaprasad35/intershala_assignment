# DeepThought — Business Analytics Internship Assignment

This repository contains deliverables and tooling for the DeepThought Target Company Research assignment.

Structure:
- `data/` — CSV and seed lists
- `scripts/` — scraping and processing utilities
- `outputs/` — generated company CSV and artifacts
- `methodology.md` — research methodology and sources
- `proposal/1000_company_proposal.md` — Part B proposal

Next steps:
1. Populate `data/seed_candidates.txt` with candidate company names (or allow the scraper to query sources).
2. Run `scripts/scrape_companies.py` to fetch pages and extract evidence (requires network and Python deps).
3. Review `outputs/companies.csv` and finalize the 25 selected companies.
